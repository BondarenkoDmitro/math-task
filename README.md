# 📊 Math Task - Теорія множин

## 📝 Опис завдання

**Завдання №2 — Варіант 3**

Веб-додаток для розв'язання задач з теорії множин. Програма реалізує:

1. **Операцію різниці множин** (A \ B)
2. **Бінарне відношення** ρ ⊂ A × B
3. **Матрицю бінарного відношення**

## 🎯 Варіант завдання

**Операція:** A \ B (різниця множин)

**Відношення:** ρ = {(a,b) | a ∈ A & b ∈ B & (a+b) ⋮ 2}
- Тобто пари (a,b), де сума a+b ділиться націло на 2 (є парним числом)

## 🔍 Приклад роботи

**Введення:**
- Множина A: {1, 3, 5, 7}
- Множина B: {2, 4, 6}

**Результат:**
- A \ B = {1, 3, 5, 7} (оскільки жоден елемент A не міститься в B)
- Бінарне відношення ρ: {(1,3), (3,1), (5,1), (5,3), (7,1), (7,3)}
  - (1,3): 1+3=4 ⋮ 2 ✓
  - (3,1): 3+1=4 ⋮ 2 ✓
  - і т.д.

## 📚 Додаткова інформація

### Математичні основи:
- **Різниця множин A \ B** = {x | x ∈ A і x ∉ B}
- **Бінарне відношення** — це підмножина декартового добутку A × B
- **Матриця відношення** — це двовимірна таблиця, де 1 означає наявність зв'язку, а 0 — відсутність
